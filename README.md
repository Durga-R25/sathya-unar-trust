# Sathya Unar Charitable Trust - Educational Technology Projects

This repository contains educational technology applications and tools developed for Sathya Unar Charitable Trust to support rural education initiatives.

## 📚 Projects

### 1. [ScienceVerse - District Science Competition Platform](./scienceverse/)

A TikTok-style mobile-first platform for conducting district-level science competitions in rural India, designed for 20,000+ students (Class 7-9) with 5,000 concurrent users.

**Key Features:**
- 📹 Video upload and evaluation system
- ⭐ Multi-tier evaluation (Judges 70%, Teachers 20%, Students 10%)
- 🏆 Leaderboards and analytics
- 🔥 TikTok-style swipeable feed
- 🎯 Categories: Physics, Chemistry, Biology, Environment, Technology

**Tech Stack:** React, Firebase (Authentication, Firestore, Storage, Cloud Functions)
