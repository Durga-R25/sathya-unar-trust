## Video Checklist — What to Verify Before Using

For each video, confirm:
- [ ] Official government/KalviTV channel (not third-party)
- [ ] Correct class (8 or 9) and unit number
- [ ] Full lesson covered (not just partial)
- [ ] Audio clear in Tamil
- [ ] Video quality adequate (720p minimum preferred)
- [ ] No copyright issues / ads-free or minimal
- [ ] Video ID extractable (format: youtube.com/watch?v=XXXXXXXXX)

## Video Embedding Rules

YouTube embed URL format for the app:
```
https://www.youtube.com/embed/{VIDEO_ID}?
  start={checkpoint1_seconds}
  &rel=0
  &modestbranding=1
  &cc_load_policy=1
```

Parameters:
- `rel=0` — Don't show related videos after lesson
- `modestbranding=1` — Minimal YouTube branding
- `cc_load_policy=1` — Show Tamil captions if available
- `start=` — Skip to specific second if intro is long
